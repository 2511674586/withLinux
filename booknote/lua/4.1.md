## Introduction to C API of Lua

Use Lua in the C programming language.  

For application example see `lua.c`, for library example see `lmathlib.c`, `lstrlib.c` and so on.

A virtual stack is used to eliminate the gap between C and Lua  
1. Lua uses Garbage Collection (GC), while C requires freeing memory explicitly.  
2. Lua uses dynamic type, while C uses static type.  

A simple lua interpreter [mylua.c](./codes/mylua.c)
